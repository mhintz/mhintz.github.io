About Me

I'm Mark Hintz, a creative coder at Periscopic, a socially-conscious data visualization firm. I live in Portland, Oregon, and I build interactive tools that help users explore interesting stories found in datasets.

My proficiencies include a strong grasp of Javascript and HTML5, and I can do backend development in PHP or Python. I can make a pixel-perfect implementation of any design, with flexible object-oriented architecture and readable code. I'm experienced at building apps from scratch using a modular, scalable Javascript architecture that incorporates data I/O, extensive user interaction, complex client-side data filtering, view updating and switching with lively motion and transitions, and accurate visual scales that precisely depict data quantities and differences.

In general, I'm interested in creative visual and interactive applications of web technology, data visualization, interactive visual art, and installations. In my spare time programming, I explore Processing, Cinder, OpenGL graphics, and strange geometries. On my Github page, you can find code for some of my side projects.
